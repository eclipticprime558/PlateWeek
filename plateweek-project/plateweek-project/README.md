# PlateWeek 🍽️

**Meal & Workout Planner** — single-file web app, no server required.

## Quick Start
1. Open `index.html` in Chrome
2. That's it — no install, no build step

## Features
- 📅 **Week & Day views** — plan meals across a 7-day grid
- ✨ **Planning Wizard** — "Plan This Week" / "Plan Tomorrow" modal with pattern-based suggestions
- 🛒 **Grocery List** — auto-generated from your meal plan, organized by store aisle order
- ⇅ **Reorderable aisles** — drag or use arrows to match your actual store layout, saved automatically
- 💪 **Workout tracking** — 8 workout types, duration, notes, completion streak
- 🎙 **Voice commands** — say "Plate Week, add eggs" to add to grocery list hands-free
- 🔔 **Check-in notifications** — reminders after each meal window
- 📊 **Pattern recognition** — suggests meals based on your history, prevents back-to-back repeats

## Voice Commands
Say **"Plate Week"** to wake the mic, then:
- `"add eggs"`
- `"add chicken breast to the list"`
- `"add frozen berries"`

> Voice requires Chrome browser

## Storage
All data stored in browser `localStorage`. No account, no server, no internet required after first load (fonts load from Google Fonts on first open).
